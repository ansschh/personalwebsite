Overview

The problem we are aiming to solve is an issue that often shows up during A.I development for startups and university labs. For machine learning, developers need data to be labelled for the program to understand what the data is. However, this necessary process is extremely tedious and lengthy as data that requires human labellers are sophisticated data with complicated imaging. By crowdsourcing the data and having a large quantity of labellers, we will be able to solve this problem for AI development by shortening the time and cost.

we are providing data labelling services to startups and university research to scale the development of AI, we have our own data labelling software which has very easy UI and tools to label the data that's why we can able to crowdsource the data which go multiple verification steps with multiple labellers to increase the accuracy of data labelling process with short amount of time.

Aims

To make medical image labeling mor faster cheaper accurate to speed up advancement in medical science.

The outcome

Through trying various image labeling techniques, and trying various data verification techniques we have been able to achieve a 95% accuracy in our data labeling method and were able to reduce the actual cost of AI production my 60%.
